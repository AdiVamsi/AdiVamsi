## What I Build

**Async Backend Services** — Python and Java/Spring Boot services processing data-driven workflows at scale. Clean API contracts, 99.9%+ uptime, observable and debuggable in production.

**AI Workflows & LLM Pipelines** — Production LLM API integration, prompt-engineered classification, RAG systems, and agentic orchestration. Built to eliminate manual work from core business processes, not to pass benchmarks.

**Agent Systems** — Self-improving platforms where agents autonomously run optimization loops — program → artifact → metric → commit. Multi-agent coordination with pluggable LLM providers.

**Operator Products** — CRMs, lead engines, and real-time dashboards designed for operators running daily workflows. WhatsApp capture, AI scoring, live updates via WebSockets.

**Indian SME Engine** — LLM-powered multi-tenant CRM for Indian SMBs. Captures leads from web forms and WhatsApp, auto-classifies and scores them via AI, surfaces prioritized follow-ups through a real-time operator dashboard. Full backend: JWT auth, multi-tenant data isolation, WebSocket live updates, AI scoring engine. 100+ commits, actively developed.

**Agent Hub** — Self-improving agent platform where each agent autonomously optimizes a real-world problem via a 3-primitive loop: `program.md → artifact → scalar metric → git commit or reset`. Runs ~12 experiments/hour. Meta-agent coordination layer enabling hands-off overnight optimization loops — pluggable into Claude, Codex, or any LLM provider.
